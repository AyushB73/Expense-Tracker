BUTTON DETAILS:
───────────────

Location:     Transaction History section, top right
Color:        🔴 RED (to distinguish from green CSV button)
Icon:         📄 PDF file icon
Text:         "Export PDF"
Position:     Right next to "Export CSV" button


VISUAL APPEARANCE:
──────────────────

┌──────────────────┐  ┌──────────────────┐
│ 📥 Export CSV    │  │ 📄 Export PDF    │  ← This one!
└──────────────────┘  └──────────────────┘
    (Green)               (Red)


WHAT HAPPENS WHEN CLICKED:
───────────────────────────

1. Button is clicked
2. PDF is generated in browser
3. File automatically downloads
4. Success notification appears
5. PDF opens in your default PDF viewer


FILE DOWNLOAD LOCATION:
───────────────────────

Windows:    C:\Users\[YourName]\Downloads\
Mac:        /Users/[YourName]/Downloads/
Linux:      /home/[YourName]/Downloads/

(Or wherever your browser is configured to save downloads)
